# A Personalized Echo

Write a Bash script which accepts **name** as input and displays the greeting "Welcome (name)"

## Input Format

```abc
There is one line of text, **name** .
```

## Output Format

```shell
One line: "Welcome (name)" (quotation marks excluded).
The evaluation will be case-sensitive.
```

## Sample Input 0

```abc
Dan  
```

## Sample Output 0

```abc
Welcome Dan  
```

## Sample Input 1

```abc
Prashant
```

## Sample Output 1

```abc
Welcome Prashant
```
